Problemas comunes

  • No se encontraron fotos: Verifique que la carpeta de origen contenga formatos compatibles (JPG, HEIC, RAW, etc.)
  • Ciudad detectada incorrecta: Descargue la base de datos específica del país mediante "Gestionar base de datos de países"
  • La aplicación se bloquea: Asegúrese de tener al menos 200 MB de espacio libre para operaciones de base de datos
